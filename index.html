What is Migblock?

Simply put, Migblock is a passion project; An infinite Voxel Universe wherein which you can do just about anything your heart desires.

Migblock features large worlds with near infinite build height, as well as depth, with the ability to create your own Voxel Starships to traverse an infinite Procedural Galaxy littered with all sorts of Procedural Planets. - and perhaps something a little spooky also.



Realistic Item physics and a powerful dynamic lighting engine.



Custom wings and support for HD Player Skins. - So you can express yourself however you like.


There are lot's of minerals and such with the ability to create custom Tools.


The world is alive with a dynamic and simulated ecosystem.
Block updates will occur regardless of which Chunks are loaded.
- Seasons also roll across the world.


With Multiplayer support you don't have to be so alone.
The Server software is bundled with the Game, there's LAN support too.
We've also a public community server where all are welcome.

Migblock is currently in early access. So not all of these things have found their way to the full release as yet. Some things might change or be improved, so nothing's set in stone. You've a chance now to join us on this adventure and help create the prefect Game.


Watch this space for updates!
